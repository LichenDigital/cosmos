cosmos web UI

The cosmos web UI is a toolset for building responsive, modern, and beautiful web user interfaces. It utilizes SASS (compiled to CSS), javascript, and SVGs to provide elegant components for building web applications. Eventually it will be integrated into cosmos's backend and universal UI protocol.

cosmos is: * modular * responsive * beautiful * mobile first * lightweight * fast * scalable * customizable * modern * visual styles * box-model * flex-box



controls


Controls are interactive component containers that are comprised of one or more smaller control elements.
Interactive elements can be added or removed from a control, and can be programmed to behave differently while in a control. Control elements do not do anything by themselves (at least so far). Control elements are elements that can exist autonomously and are familiar HTML elements like buttons, inputs, select (dropdowns), and labels. Some elements like labels are more robust than they have traditionally been used for in HTML. They also behave and appear differently based on if they are contained within a another element like a button.
